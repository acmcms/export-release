• General

    'vfs' object providing means to access and manipulate internal virtual file system.

  • Hierarchy

    Directory hierarchy.

    • Root folders:
      • /private - read-write instance local (private) data
      • /cache - read-write instance local (private) cache for shared, cluster and/or public data
      • /shared - read-write machine local shared data, mapped to real file-system and used for interprocess communication, available to all running services
      • /protected - read-write cluster data, shared/accessible from any cluster machine
      • /public - read-only public data
      • /union - read-only local, cache, shared, cluster and public data mixed altogether
      • /local - read-write instance local (private) data
      • /cache - read-write instance local (private) cache for cluster data
      • /cluster - read-write cluster data
      • /shared - read-write shared cluster-wide data available to all running services
      • /public - read-only public data
      • /union - read-only public, local, cache, shared, cluster data mixed altogether
    • Each of root folders contains:
      • /home - per user data (as in sub-folder for every user)
      • /resources - permanent data which is supposed to change on system updates/upgrades only.
        • /lib - CommonJS libraries
        • /skin - skins
      • /settings - variable data which is supposed to change during normal operation.
        • /manual - manuals descriptors definition files
          • /developer - manuals for developers (programmers, coders)
          • /maintainer - manuals for maintainers (system administrators)
        • /periodic - periodic tasks descriptions for 'periodic' module.
          • /hourly
          • /daily
          • /weekly
          • /monthly
        • /service - daemons, service descriptions for 'service' module.
        • /shell - global shell commands definition files
        • /test - test tasks definition files
        • /web - web related definition files
          • /hosts - web shares definition files.
