Каждый язык создает Code (ProgramPart / Instruction) из исходного текста.
Каждому языку соответствует имя, расширения файлов, набор mime-типов.

Сама система Evaluate для каждого своего зарегистрированного в ней языка
регистрирует factories для получения Code по имени языка, каждого расширения 
файлов и mime-типа.

Для получения объектов определенного типа из исходников на разных языках, для 
каждого такого "определенного типа" требуется создать factory которая будет
создавать экземпляры этого определенного типа объектов из Code полученный 
средствами системы Evaluate (методами Evaluate, а не factories языков).
