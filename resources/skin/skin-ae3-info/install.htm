• More specific types of installation are:
    • ae3bsd embedded software for dedicated (could still be virtual) i386 or x86-84 system.
    • ae3win installation instructions for Microsoft Windows systems.
    • ae3deb installation instructions for Debian (GNU/Linux) systems.

  • Requirements

    • Installed and working JAVA SE 7
    • Ability to open not less than 4096 files for java process
    • Ability to start not less than 512 threads for java process
    • Unlimited max thread execution time for java process
    • Not less than 512 megabytes allocated for java process
    • 100% JDBC-compatible database and driver

  • Database specifics

    System is bundled with drivers for:

    • mysql
    • MS SQL
    • postgres
    • oracle
