debug.js

Debug output console for easier script development.

// Usage like this: // window.debug && debug( <message> ); // or // top.debug && top.debug( <message> );
  • debug - script source code (for studying the code and comments).
  • debug - minified location (for actual usage in a web page).

What does it look like?

When collapsed it looks like small gray paragraph sign in a small light gray box in one of the window corners.


fig1: console collapsed.

When you click on it - it expands to a half-window-width red colored console.


fig2: console expanded.

This console have whole lot of controls to collapse it back, change transparency, clear output, choose one of the window corners, pause output (console becomes grey as well) and 'grep' (filter) output messages by a text filter string.


fig3: console paused, transparent and docked top-left.
