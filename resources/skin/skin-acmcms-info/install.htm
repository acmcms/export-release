Installation

  • More specific types of installation are:
    • acmbsd embedded software for dedicated (could still be virtual) i386 or x86-84 system.
    • acmwin installation instructions for Microsoft Windows systems.
    • acmdeb installation instructions for Debian (GNU/Linux) systems.

  • Requirements

    • Installed and working JAVA SE 6.0
    • Ability to open not less than 4096 files for java process
    • Ability to start not less than 512 threads for java process
    • Unlimited max thread execution time for java process
    • Not less than 256 megabytes allocated for java process
    • 100% JDBC-compatible database and driver

  • Database specifics

    System is bundled with drivers for:

    • mysql
    • MS SQL
    • postgres
    • oracle
