        java.lang.String fixLocation(ExecProcess process,
                           java.lang.String path,
                           boolean absolute)
        Converts control tree location to an external URL considering share setups, access types and so on.
        Parameters:
        process - current context
        path - control tree location (/aa/bb/vvxxv/....)
        absolute - true: doesn't consider current context location - same result for any share used for current request.
        false: tries to make location on the same share as share used for current request.

        string
