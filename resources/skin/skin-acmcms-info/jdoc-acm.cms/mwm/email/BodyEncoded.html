    public abstract class BodyEncoded
extends Body
    This is the abstract base class for all mail bodies containing a content which must be encoded before transport. This could be attachments, images, text or binary files.

    The class uses a Encoder object to prepare the body for transport. The Encoder is an abstract skeleton which developers can subclass to provide their own custom transport encodings.

    This body part contains a header object which manages the header fields. The contructor will attempt to set one header field automaticly: The "Content-Transfer-Encoding" field. The value of this field is fetched from the Encoder object's getContextName method. Implementors who whishes to set this field to something else can retrieve the header object through the getHeader method and set the field manually.
