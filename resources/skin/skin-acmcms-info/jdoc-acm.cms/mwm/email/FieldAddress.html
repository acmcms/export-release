    java.io.Serializable

    public final class FieldAddress
extends FieldSimple
    This is a dedicated header field which extends the EmailHeaderField for addresses including header field encoding possibilities. Encoding is performed by a Encoder object which can be specified upon creation of a new address field object. Default encoding is the so called "q" encoding using the "ISO-8859-1" character set.

    Multiple addresses will be separated by a ",", a canonical line break and a tab, this behavior is called "folding" and is presented in RFC 822.
