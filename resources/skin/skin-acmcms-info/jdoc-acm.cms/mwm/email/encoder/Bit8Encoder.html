mwm.email.encoder

Class Bit8Encoder

  • java.lang.Object
    • mwm.email.encoder.Encoder
      • mwm.email.encoder.Bit8Encoder

  • public final class Bit8Encoder
extends Encoder
    This encoder ensures that the encoded content only will be sent directly to the stream without any encoding. The toByteArray will attempt to convert the characters to the given charcter set but regardless this - the encode method will only write the single bytes in without any encoding or preparation.

    If no charset is provided before the toByteArray method is called the mentioned method will return an array of the low 8 bits of every unicode character provided in the method parameter.

    This class should not be used so much by "message creators" as by "message parsers". If characters outside the "US-ASCII" character set is needed a upon creation of message a "Quoted-Printable" or a "Base64" encoding should be used. Message parsers might however find this encoder useful since it will not distort any characters using the last bit in a byte, thus it can be used for textual content where the character set is not defined but can contain illegal characters due to faulty message creation.
