    public class Base64Encoder
extends Encoder
    This Encoder performs converting for byte arrays and strings in the Base 64 encoding. The Base64Encoder can be created with a charset value to perform convertion from characters to bytes according to a specified character set encoding in the toByteArray method. The charset can be validated with the validateCharset method which will tell if the charset is recognised and usable. If no charset is provided before the toByteArray method is called the encoder will asume the text is in "US-ASCII".
