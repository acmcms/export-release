    B_Decoder
    Decoder object for base 64 content.
    B_Encoder
    This Encoder performs converting for byte arrays and strings in the "b" encoding used in email header fields.The B_Encoder must be created with a charset value to perform convertions from characters to bytes in the toByteArray method.
    Base64Decoder
    This class extends the Decoder to provide decoding for the base 64 encoding.
    Base64Encoder
    This Encoder performs converting for byte arrays and strings in the Base 64 encoding.
    BinaryEncoder
