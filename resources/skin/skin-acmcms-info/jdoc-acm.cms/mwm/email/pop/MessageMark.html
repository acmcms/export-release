mwm.email.pop

Class MessageMark

  • java.lang.Object
    • mwm.email.pop.MessageMark

  • public class MessageMark
extends java.lang.Object
    This class is a simple utility used by the POPRetriever and hold number and size of pop server messages. It can either be returned by the POPRetriever as total number and size of maildrop as an answer on the "status" command or as the number and size of a specific message in the maildrop after a "list" command.

    Lars J. Nilsson
