    java.io.Serializable

    public class BodyParallel
extends BodyMultipart
    This class represents a parallel multipart body according to the MIME message standard. The parallel body part contains different parts of content that is considered to be of equal precedence and should be displayed parallel. Not many client handles this body part and it is not widely used.

    This class requires a boundary string upon creation. See the documentation of the MultipartBody class for more information.

    This class will set one header field by default, the "Content-Type" header which will be set to "multipart/parallel". This header field should not be changed in most cases.
