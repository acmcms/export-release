    public final class BodyRelated
extends BodyMultipart
    This class represents an related multipart body according to the MIME message standard.

    See the MultipartBody documentation for a discussion of boundaries.

    This class will set one header field by default, the "Content-Type" header which will be set to "multipart/related". This header field identifies the type of body is a MIME context and should not be changed,
