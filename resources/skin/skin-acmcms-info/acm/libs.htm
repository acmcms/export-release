Developer Tools: CommonJS libraries

CommonJS libraries - objects accessible through thread's global context provided by system as an execution environment helpers, factories and API's.

  • Usage

    There is require function available in global context. You call it like require( __package_name__ + '/' + __module_name__ ) to load the module specified and get an instance of API for module loaded.

    Module will be searched in following locations:

    1. server's lib/ folder
    2. acmcms instance's %protected% resources/lib/ folder
    3. acmcms %public% resources/lib/ folder
    Module file is a '.js' file with module name placed in folder with package name. For example, ru.acmcms/dbi module is located in %public%/resources/lib/ru.acmcms/dbi.js file.

    Example: <%FORMAT: 'js' %> var dbi = require('ru.acmcms/dbi'); dbi.executeTransaction(.....); <%/FORMAT%>
